# Локальный запуск проекта
1. Склонировать проект

````
git clone https://github.com/dezzerlol/finance-frontend
````


2. Перейти в папку
   
````
cd finance-frontend
````

3. Установить зависимости
   
````
 yarn или npm install
````

4. Собрать и запустить проект
   
````
yarn build
yarn preview
````

5. Проект доступен на

````
http://localhost:3000/
````
